# Diular – Bastion du Froid

**Résumé :** Cité-campement au sud. Garnison, palissades, échanges avec les nains.

## Lieux

On y trouve le **Fortin de Givre**, la **Halle de Troc** où se font les échanges avec les nains, et les **Feux Longs**, postes de veille contre la banquise.
